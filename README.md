# Actividad 3
 Juego Pacman

 Para esta actividad necesitamos aprender a hacer que la interfaz gráfica mostrara un camino, lo cual fue mucho más complicado que en la acrividad pasada. Aún así, pudimos comprender cada función y modificar el juego para que los puntos rojos (que representan a los fantasmas en el juego original) se movieran más rápido, y así incrementar la dificultad del juego.
